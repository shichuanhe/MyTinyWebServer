        a.在构造函数中申请分配资源，在析构函数中释放资源。因为C++的语言机制，当一个对象创建时会调用构造函数，当对象超出作用域时会自动调用析构函数。所以在RAII指导下，我们应该用类来管理资源，将资源和对象的生命周期绑定。
